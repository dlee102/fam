# 뉴스 백테스트·호라이즌 분석: 진입 시점 정의 (분 단위)

`data/publish_horizon_curve.json` 및 `scripts/entry_hold_analysis.py`에서 쓰는 진입 라벨 **A~F**를, **가격이 어떤 시각·구간에 해당하는지** 분 단위로 정리한 문서입니다.

## 공통 전제

| 항목 | 내용 |
|------|------|
| 시세 | EODHD **5분봉**만 사용 (일봉 OHLC 숫자는 미사용) |
| 장중 봉 | `datetime` 문자열의 UTC 시각이 **00:00~06:25**인 봉만 장중으로 간주 → **KST 09:00~15:25** 구간과 대응 (`entry_hold_analysis.py` `_is_market_bar`) |
| 5분봉 의미 | 봉 시각은 **구간 시작**. **종료 시각** = 시작 + **5분** (`fivem_end_utc_naive`) |
| 기사 시각 | API **`published_at`만** 사용. 타임존 있으면 UTC로 환산, **없으면 KST**로 해석 (`analyze_10m_return_path.py` `parse_publish_utc_naive`) |
| 이벤트일 T0 | 기사 **KST 달력일** `t0`. 주말·휴장이면 **그다음 첫 거래일**이 실제 세션으로 맞춰짐 (`eod_index_on_or_after`) |

### 한국장 정규장 5분봉 (KST 기준, 하루당 개략)

- **첫 봉**: 09:00 시작 → **09:05에 종가 확정** (1번째 5분봉 종료)
- **둘째 봉**: 09:05 시작 → **09:10에 종가 확정**
- **마지막 봉**: 15:20 시작 → **15:25에 종가 확정** (당일 장중 마지막 5분봉)

## 진입별 시점 (분 단위)

### A — `T=0` **당일 장 종가** 진입

- **의미**: 이벤트에 해당하는 **첫 거래일**(`t0` 이후 첫 세션)의 장중 **마지막 5분봉 종가**
- **KST 시각**: 해당 거래일 **15:25**에 확정되는 종가 (마지막 봉의 close)
- **코드**: `last_intraday_session_close(all_intra, t0_date)` (`entry_hold_analysis.py`)

---

### B — `T+1` **첫 5분봉 시가** 진입

- **의미**: T0 세션의 **다음 거래일**(`bars[i0+1]`) 장중 **첫 5분봉의 시가**
- **KST 시각**: 그날 **09:00** (첫 봉이 09:00~09:05이므로 시가는 장 시작 시각)
- **코드**: `first_intraday_session_open(all_intra, t1_date)`

---

### C — `T+1` **장 종가** 진입

- **의미**: **다음 거래일** 장중 **마지막 5분봉 종가**
- **KST 시각**: 그날 **15:25** 확정 종가
- **코드**: `last_intraday_session_close(all_intra, t1_date)`

---

### D — `T+1` **첫 5분봉 종가** 진입

- **의미**: **다음 거래일** 첫 5분봉의 **종가**
- **KST 시각**: **09:05** (09:00~09:05 봉이 끝나는 시각)
- **코드**: `t1_first_close_second_open` → 첫 봉 `close`

---

### E — `T+1` **두 번째 5분봉 시가** 진입

- **의미**: **다음 거래일** 두 번째 5분봉의 **시가**
- **KST 시각**: **09:05** (09:05~09:10 봉의 시가)
- **코드**: `t1_first_close_second_open` → 둘째 봉 `open`

---

### F — **공개 시각 직후** 첫 유효 5분봉 **종가** 진입 (기사별로 시각 가변)

- **의미**: `published_at`을 UTC naive로 바꾼 뒤, 장중 5분봉을 시간순으로 보면서  
  **봉 종료 시각 ≥ 공개 시각**인 **가장 이른** 봉의 **종가**
- **KST 시각**: 기사마다 다름. 예시:
  - 공개가 **09:02 KST** → 첫으로 끝나는 봉은 09:00~09:05 → 종가 확정 **09:05**
  - 공개가 **09:10 KST** → 09:05~09:10 봉이 첫 조건 충족 → 종가 확정 **09:10**
  - **장 마감 이후** 공개(예: 저녁) → **다음 거래일** 장중에서 같은 규칙으로 첫 봉 선택 (`first_close_after_publish_on_calendar`)
- **코드**: `first_close_after_publish_on_calendar` + `eod_index_for_session_ymd` (`entry_hold_analysis.py` 141~181, 254~267행)

---

## 보유 일수와 청산 시각 (참고)

- **청산 가격**: (진입이 정해진 뒤) **N거래일 뒤** 해당일 장중 **마지막 5분봉 종가** → 그날 **15:25 KST** 확정과 동일한 규칙
- **A·F**: 진입일이 잡힌 날의 EOD 인덱스 기준 `+N`
- **B~C~D~E**: 진입이 `T+1`에 있으면 그 **진입 거래일** 기준 `+N` (`entry_hold_analysis.py` 상단 docstring, 275~306행)
