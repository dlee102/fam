| 데이터 소스 | `data/publish_horizon_curve.json` |

| 집계 단위 | 기사 1건 × 종목 1코드 = 표본 1개 |

## 갱신 방법

최신 데이터로 다시 채우려면 프로젝트 루트에서:

```bash
python3 scripts/entry_hold_analysis.py
```

실행 후 `data/publish_horizon_curve.json`의 `generated_at`과 `entries`를 확인하고, 이 문서의 표를 그에 맞게 업데이트하면 됩니다.
