# 팜이데일리(Pharm Edaily) 연동 및 AI 뉴스 분석 플랫폼 기획안

## 1. 개요
본 기획안은 '팜이데일리(Pharm Edaily)'의 프리미엄 바이오/제약 뉴스 데이터를 FAM(Financial Analysis Module) 플랫폼에 통합하고, 이를 AI 기술로 분석하여 투자자들에게 차별화된 인사이트를 제공하는 시스템 구축을 목표로 한다.

## 2. 추진 배경 및 목적
*   **정보의 비대칭성 해소**: 바이오/제약 분야는 전문 용어가 많고 임상 결과 해석이 어려워 일반 투자자들이 접근하기 힘든 영역임.
*   **프리미엄 데이터 활용**: 팜이데일리에서 제공하는 심층 취재 및 단독 기사를 데이터화하여 분석 가치를 극대화함.
*   **실시간 대응 체계 구축**: 뉴스 발생 시 해당 종목의 퀀트 점수(수급, 거래량 등)와 연계하여 즉각적인 투자 판단 근거를 제공함.

## 3. 주요 기능 구성

### 3.1. 자동화된 데이터 수집 (Data Ingestion)
*   **실시간 크롤링 엔진**: `pharm_crawler`를 고도화하여 팜이데일리 프리미엄 기사를 실시간으로 수집.
*   **세션 관리**: Chrome 프로필 기반 로그인 세션을 유지하여 유료 기사 본문 전체를 안정적으로 확보.
*   **증분 수집**: `manifest_per_article.json`을 통해 중복 수집을 방지하고 신규 기사만 효율적으로 업데이트.

### 3.2. AI 기반 뉴스 분석 (AI Analysis)
*   **종목 매핑 (Entity Linking)**: 기사 본문에서 언급된 바이오 기업 및 관련 티커(Ticker)를 정확히 추출.
*   **임상 단계 분석**: 임상 1/2/3상, IND 승인, NDA 신청 등 주요 마일스톤을 분류하고 성공 가능성 및 시장 영향력을 평가.
*   **감성 분석 (Sentiment Analysis)**: 기사의 톤(긍정/부정/중립)을 분석하여 시장의 심리적 지표로 활용.
*   **요약 서비스**: 긴 전문 기사를 3줄 핵심 요약 및 투자 포인트로 변환하여 가독성 증대.

### 3.3. 퀀트 데이터 결합 (Quant Integration)
*   **뉴스-수급 연동**: 뉴스 발생 시점 전후의 외국인/기관 수급 변화(`foreign_flow`)를 시각화.
*   **실시간 점수 산출**: 뉴스 임팩트를 점수화하여 기존 FAM 퀀트 스코어에 가중치로 반영.
*   **이상 징후 탐지**: 뉴스 발표 전 비정상적인 거래량 폭증이나 호가 불균형 발생 시 알림 제공.

## 4. 시스템 아키텍처

### 4.1. 백엔드 (Node.js/Python)
*   **Python**: 크롤링, NLP 분석, 퀀트 스코어 계산 등 연산 집중적 작업 수행.
*   **Node.js (Next.js API Routes)**: 프론트엔드 요청 처리 및 데이터 서빙.

### 4.2. 프론트엔드 (React/Next.js)
*   **대시보드**: 실시간 팜이데일리 뉴스 피드와 해당 종목의 차트/퀀트 점수를 한 화면에 배치.
*   **바이오 특화 뷰**: 임상 일정표(Calendar), 파이프라인 현황판 등 바이오 섹터 전용 UI 제공.

## 5. 기대 효과
*   **투자 의사결정 속도 향상**: 복잡한 바이오 뉴스를 AI가 즉시 해석해줌으로써 빠른 대응 가능.
*   **데이터 기반 투자**: 단순 루머가 아닌 실제 뉴스 데이터와 수급 데이터를 결합한 객관적 지표 활용.
*   **플랫폼 경쟁력 강화**: 팜이데일리라는 독점적 콘텐츠를 FAM 플랫폼의 핵심 기능으로 내재화.

## 6. 향후 로드맵
1.  **Phase 1 (1개월)**: 크롤러 안정화 및 기사-티커 매핑 로직 완성.
2.  **Phase 2 (2개월)**: LLM(Large Language Model) 연동을 통한 기사 요약 및 임상 단계 자동 분류 도입.
3.  **Phase 3 (3개월)**: 뉴스 임팩트 기반 백테스팅 시스템 구축 및 자동 매매 전략 연계.
